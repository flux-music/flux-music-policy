📌 1. Introduction

This Privacy Policy explains how Flux Music (“the Bot”) collects, uses, and protects user data.

By using Flux Music, you agree to the collection and use of information in accordance with this policy.

---

📊 2. Data We Collect

Flux Music collects only the data necessary for proper functionality:

- Server IDs
  Used for server-specific settings such as prefixes and 24/7 mode

- User IDs
  Used for:
  
  - User profiles (command usage statistics)
  - Playlists
  - Blacklist system
  - No Prefix access
  - Vote Bypass access
  - Spotify account linking

- Command Usage Data
  Tracks how many commands are used (no message content is stored)

- Music Activity Logs
  Includes player events such as:
  
  - Player create/destroy
  - Autoplay events

---

🗄️ 3. Data Storage

- Data is stored using SQLite3
- Data is stored permanently unless deletion is requested
- Storage is limited to what is required for bot functionality

---

🔍 4. How We Use Data

Collected data is used to:

- Provide and maintain bot functionality
- Enable features such as playlists, profiles, and prefixes
- Improve performance and user experience
- Enforce rules (e.g., blacklist system)
